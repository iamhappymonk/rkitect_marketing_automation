# rkitect.ai — brand brief (synthesised for mem0 ingest)

## Product

rkitect.ai (codename Render Studio). AI-coordinated studio of specialist agents that generates photoreal interior renders for architects in under 60 seconds at $0.20 per render. Pre-launch, beta cohort of 14 NYC studios. Built by an architect after losing pitches to render queues.

## Buyer

Primary: small architecture studios (2–15 people), boutique principals.
Secondary: mid-size firms (15–75 people), viz managers, innovation leads.
Tertiary: real-estate, proptech, virtual staging, listing ops.

Project types served: residential · commercial · hospitality · outdoor.

## The studio (specialist agents — architect language)

- **Chief Architect** — reads the brief, routes work, synthesises the response.
- **Interior Designer** — material thesis, palette, lighting strategy, style references.
- **Space Planner** — room programme, scaled floor plans, adjacency validation.
- **3D Artist** — isometric, room geometry, furniture placement at scale, CAD export.
- **Renderer** — photoreal output, multi-view, variations from one comprehension pass.
- **Edit specialists** — Material, Lighting, Furniture, Layout. Surgical small-region edits without re-rendering the room.

## Discovery flow (3 phases before any render)

1. **Context** — site diagram + type-specific brief in your voice.
2. **Precedents** — small set drawn to brief; pick 1–2 anchors.
3. **Direction** — bubble diagram + parti sketch, refined before render spend.

## Thesis Deep Mode

Every thesis composed against a 42-point rubric (palette, programme, lighting, precedent justification, materials defensibility). A second pass scores. Below 80 → auto-refine and retry before user sees output. Last review score 87/100.

## Memory

Studio voice + vetoes + wins persist across projects. New brief opens with prior decisions already on the table.

## Brand voice

Considered. Quiet. Authoritative. Anti-jargon. Architect language, not engineer language. No Canva-gradient moves. Brand red ≤5% visual weight. Architectural ink-blue is the informational accent. Spectral display + Switzer body + Geist Mono labels.

## Compare math (vs traditional)

Traditional: 3 days, $300–1,200 per cycle (3ds Max + render farm + viz seat).
rkitect.ai: 60 seconds, $0.60 per cycle (5 agents in-app).
