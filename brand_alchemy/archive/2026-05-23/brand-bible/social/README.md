# social/ — Social Media Content

This folder contains social media posts, threads, and community content for all platforms.

## What Goes Here

- LinkedIn posts and long-form content
- Twitter/X posts and threads
- Instagram captions and story content
- TikTok scripts (if applicable)
- Facebook posts
- Community post collections (by topic)
- Social media calendar and planning docs
- Social media campaign docs
- Engagement templates and response templates
- Platform-specific strategies

## Subfolder Structure (When Needed)

```

```

## File Naming Convention

- `{platform}-post-{topic}-{date}.md`
- `{platform}-thread-{topic}-{date}.md`
- `{topic}-social-calendar-{month}.md`
- `{campaign}-social-plan.md`

## Social Post Format

For individual posts, include:

- Post title/topic
- Full post copy
- Hashtags (if applicable)
- Visual assets needed
- Target audience/platform
- CTAs and links
- Engagement goal

## Social Thread Format

For threads, include:

- Thread title
- Complete thread text (numbered by tweet)
- Key message/summary
- Platform considerations
- Visual assets if any
