# Brand Bible Quick Start Guide

Welcome to the rkitect.ai Brand Bible. This guide helps you navigate and use the brand resources effectively.

## 🚀 Quick Start for Agents

### Step 1: Understand Brand Strategy (5 min read)

Start with these core documents in order:

1. [01-identity.md](01-identity.md) — Who we are
2. [02-positioning.md](02-positioning.md) — How we're different
3. [03-audience-personas.md](03-audience-personas.md) — Who we serve
4. [04-messaging.md](04-messaging.md) — What we say
5. [05-voice-and-tone.md](05-voice-and-tone.md) — How we sound
6. [06-visual-direction.md](06-visual-direction.md) — How we look
7. [07-go-to-market.md](07-go-to-market.md) — Where we sell

### Step 2: Load Relevant Context

Check `_context/` folder for:

- Market research
- Customer insights
- Competitor analysis
- Product details

### Step 3: Find Your Template

Check `_templates/` folder:

- Email newsletter template
- LinkedIn post template
- Landing page template
- Creative brief template
- Case study template

### Step 4: Follow the SOP

Check `_sop/` folder for:

- Approval workflows
- Brand compliance checklists
- Publishing guides

### Step 5: Save Your Output

Save final content in the appropriate channel folder:

- `ads/` — Paid advertising
- `pages/` — Website & landing pages
- `presentations/` — Pitch decks & webinars
- `reports/` — Analysis & reports
- `research/` — Market research findings
- `seo/` — Blog posts & SEO content
- `social/` — Social media posts & threads

## 🎯 Common Tasks & Where to Find Help

### I need to write a blog post

1. Check `05-voice-and-tone.md` for brand voice
2. Check `seo/README.md` for SEO guidelines
3. Use template from `_templates/` if available
4. Save to `seo/` folder

### I need to create a LinkedIn post

1. Check `04-messaging.md` for key messages
2. Use `_templates/linkedin-post-template.md`
3. Follow SOP in `_sop/social-posting-sop.md`
4. Save to `social/` folder

### I need to write ad copy

1. Check `02-positioning.md` for differentiation
2. Check `03-audience-personas.md` for tone
3. Check `_context/` for customer pain points
4. Save to `ads/` folder

### I need to brief a designer

1. Use `_templates/creative-brief-template.md`
2. Reference `06-visual-direction.md`
3. Check relevant brand docs for context
4. Save to appropriate output folder

### I need to verify a claim

1. Check `brand_alchemy/brand-bible/_context/claims-and-proof.md` (brand context folder)
2. Link to source in your output

## ✅ Quality Checklist Before Publishing

Before publishing ANY marketing content:

- [ ] Have I read the core brand documents (01-07)?
- [ ] Does my content align with our positioning (02)?
- [ ] Does my tone match voice-and-tone guidelines (05)?
- [ ] Have I included accurate claims (verified against claims-and-proof.md)?
- [ ] Have I followed the relevant SOP from `_sop/`?
- [ ] Have I used a template from `_templates/` if one exists?
- [ ] Have I saved my work to the correct output folder?
- [ ] Has this been reviewed and approved by the brand owner?

---

## 🆘 Key Rules to Remember

1. **Brand compliance is mandatory** — All outputs must align with 02-positioning.md and 05-voice-and-tone.md
2. **Approve before publishing** — Nothing goes live without human review
3. **Verify claims** — Check `brand_alchemy/brand-bible/_context/claims-and-proof.md` for public claims
4. **Use templates** — Templates ensure consistency and save time
5. **Document sources** — Link to research and brand docs that informed your work
6. **Organize outputs** — Use the right channel folder for discoverability
