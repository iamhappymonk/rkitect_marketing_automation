# Skills

Skills are reusable procedures, not brand-specific personas. They should define how work gets done and load brand context at runtime.

## Naming Rules

- Use clear workflow names.
- Keep each skill narrow.
- Avoid brand-specific assumptions.
- Reference required context files instead of copying brand facts.
