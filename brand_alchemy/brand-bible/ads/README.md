# ads/ — Paid Advertising Outputs

This folder contains ad copy, creative briefs, and campaign materials for paid advertising channels.

## What Goes Here

- Ad copy variations for different platforms
- Ad creative briefs for designers
- Campaign planning and strategy docs
- Performance data and results
- A/B test results and learnings
- Ad account structure documentation
- Platform-specific guidelines (Google Ads, LinkedIn, Meta, etc.)

## Subfolder Structure (When Needed)

```
ads/
  google-ads/
  linkedin-ads/
  meta-ads/
  other-platforms/
```

## File Naming Convention

- `{platform}-{campaign-name}-{date}.md` — e.g., `linkedin-founder-story-may2026.md`
- `{platform}-creative-brief-{campaign}.md`
- `{campaign}-performance-report-{date}.md`
