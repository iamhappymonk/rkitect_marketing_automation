# pages/ — Website & Landing Page Copy

This folder contains website page copy, landing pages, and web-based content outputs.

## What Goes Here

- Homepage copy
- Product/service page copy
- Landing page copy for campaigns
- About page content
- Pricing page copy
- FAQ page content
- Case study page copy
- SEO-optimized page variations
- Page design briefs for web team

## Subfolder Structure (When Needed)

```
pages/
  website-pages/
  landing-pages/
  case-studies/
  help-content/
```

## File Naming Convention

- `{page-name}-copy.md` — e.g., `homepage-copy.md`
- `{campaign}-landing-page.md`
- `{company}-case-study.md`
- `{topic}-faq.md`

## Page Template

Each page should include:

- Page title and meta description
- H1 heading and intro
- Body sections with clear structure
- CTA buttons and links
- Performance metrics/goals
