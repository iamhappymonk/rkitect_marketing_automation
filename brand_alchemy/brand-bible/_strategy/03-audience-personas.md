# Audience And Personas

## Primary Segments

- Independent architects.
- Small-to-mid architecture firms.
- Interior designers.
- Real estate developers and marketing teams.
- Architecture students.

## Secondary Segments

- Archviz professionals.
- BIM and Revit teams.
- SketchUp, Lumion, V-Ray, and Enscape users.
- Hospitality and retail fit-out teams.

## Persona: Studio Principal

| Attribute | Detail |
| --- | --- |
| Role | Founder, principal architect, studio owner |
| Goal | Win work and present confidently without expanding visualization headcount |
| Pain | Revisions, rendering bottlenecks, software cost, missed pitch polish |
| Trigger | Client meeting, pitch, material option request, team overload |
| Objection | AI quality, accuracy, control, and credibility |
| Best channels | LinkedIn, Instagram, referrals, demos, architecture communities |

## Persona: Working Architect Or Designer

| Attribute | Detail |
| --- | --- |
| Role | Senior architect, junior architect, interior designer, BIM lead, visualization lead |
| Goal | Create visual options and update them quickly |
| Pain | Re-rendering, Photoshop cleanup, late client changes, multi-tool fatigue |
| Trigger | Deadline pressure, review meeting, design options |
| Objection | AI may distort geometry or add review work |
| Best channels | Instagram, YouTube, Reddit, X/Twitter, peer groups |

## Persona: Architecture Student

| Attribute | Detail |
| --- | --- |
| Role | B.Arch student, thesis student, competition participant |
| Goal | Portfolio-ready renders without expensive software or hardware |
| Pain | Weak hardware, license cost, short deadlines, flat visuals |
| Trigger | Jury, thesis, NASA event, portfolio submission |
| Objection | Paid tool affordability and output reliability |
| Best channels | Instagram, YouTube, WhatsApp, Telegram, Discord, college chapters |
