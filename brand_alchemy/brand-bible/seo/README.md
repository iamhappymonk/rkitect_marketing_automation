# seo/ — Search Engine Optimization

This folder contains SEO content, keyword strategy, and organic optimization materials.

## What Goes Here

- Keyword research and strategy
- SEO-optimized blog posts
- Landing page SEO copies
- Meta descriptions and title tag strategies
- Content gap analysis
- Backlink strategy documentation
- Technical SEO notes
- Search intent analysis
- SEO performance reports

## Subfolder Structure (When Needed)

```
seo/
  blog-posts/
  keyword-research/
  landing-pages/

```

## File Naming Convention

- `blog-{topic}-{date}.md`
- `keyword-research-{focus}-{date}.md`
- `seo-{page-type}-{topic}.md`
- `seo-performance-report-{date}.md`

## SEO Content Guidelines

- Include target keyword in title and H1
- Optimize meta description (160 characters)
- Add internal links to related content
- Include word count and reading time
- Note keyword difficulty and search volume
- Link to source research when making claims
