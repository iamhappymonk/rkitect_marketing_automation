- Benchmark reports
- Data analysis and findings

- Quarterly/annual performance reports
- Competitive analysis reports
- Trend analysis reports

- `survey-results-{topic}-{date}.md`

## Report Structure

- Data/evidence
- Implications
- Recommendations
- Methodology (if applicable)
