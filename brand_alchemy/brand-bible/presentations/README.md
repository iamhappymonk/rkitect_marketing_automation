# presentations/ — Pitch Decks & Presentations

This folder contains presentation materials, pitch decks, and webinar content.

## What Goes Here

- Investor/funding pitch decks
- Customer pitch presentations
- Webinar slide outlines and scripts
- Conference talk proposals and slides
- Demo presentations
- Product walkthrough decks
- Training presentations
- Internal strategy presentations

## Subfolder Structure (When Needed)

```
presentations/
  investor-pitches/
  sales-presentations/
  webinars/
  conference-talks/
  internal/
```

## File Naming Convention

- `{audience}-pitch-deck-{date}.md` — e.g., `investor-pitch-deck-may2026.md`
- `{webinar-topic}-outline.md`
- `{conference}-talk-proposal.md`

## Presentation Guidelines

- One section per slide (roughly)
- Include speaker notes and talking points
- Specify visual assets needed (images, diagrams, etc.)
- Note any brand compliance requirements
- Time estimates for delivery
